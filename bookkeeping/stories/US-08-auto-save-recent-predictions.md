# US-08 — Auto-save recent identifications

**Persona:** Used-car shopper  
**Theme:** History & revisiting  
**Introduced:** S002

**As a** used-car shopper, **I want** the app to automatically remember my recent identifications, **so that** I can compare several listings without re-uploading the same photos.

**Acceptance criteria:**
- [ ] A successful identification is saved automatically (no extra "save" step)
- [ ] Each entry records the image (or its reference), top prediction, and timestamp
- [ ] History persists across page reloads and browser sessions (browser-local storage)
- [ ] History is stored locally only — nothing is uploaded to a server
