# S001 — Flutter Web scaffold + Playwright smoke

**Status:** planned  
**Maps to:** — (tooling validation; no product user stories yet)  
**Depends on:** [T000 — Dev container dependencies](T000.md)

## Goal

Bootstrap the `ui/` Flutter Web project and prove the Playwright → WebM pipeline works end-to-end before building DriftID inference UI. Use the **default counter app** — no API, no custom screens.

## Approach

Run `flutter create` for web, keep the stock counter template, add one stable selector on the increment button, and a Playwright script that taps it **three times** and asserts the counter reads `3`.

```
ui/                          # Flutter Web (default counter template)
  lib/main.dart              # stock counter; add Semantics on FAB
  demos/increment-counter.spec.ts
  playwright.config.ts
  package.json
```

### Proposed files

| File | Purpose |
|------|---------|
| `ui/pubspec.yaml` | Generated by `flutter create` |
| `ui/lib/main.dart` | Default counter; add `Semantics(label: 'Increment counter')` on FAB |
| `ui/demos/increment-counter.spec.ts` | Click increment 3×, assert count is 3 |
| `ui/playwright.config.ts` | WebM video, viewport, `webServer` for Flutter |
| `ui/package.json` | `@playwright/test` dev dependency |

### Flutter changes (minimal)

Keep the generated counter app. Only change needed for automation — label the floating action button:

```dart
floatingActionButton: FloatingActionButton(
  onPressed: _incrementCounter,
  tooltip: 'Increment',
  child: Semantics(
    label: 'Increment counter',
    button: true,
    child: const Icon(Icons.add),
  ),
),
```

Counter display should remain the default `'$_counter'` text so Playwright can assert on `'3'`.

### Playwright setup

Install (once `ui/` exists; Flutter and Node come from T000):

```bash
cd ui
flutter pub get
npm init -y
npm i -D @playwright/test
npx playwright install chromium   # or devcontainer postCreateCommand
```

**`playwright.config.ts`**

```ts
import { defineConfig } from '@playwright/test';

export default defineConfig({
  testDir: './demos',
  use: {
    baseURL: 'http://localhost:8080',
    viewport: { width: 1280, height: 720 },
    video: { mode: 'on', size: { width: 1280, height: 720 } },
  },
  webServer: {
    command: 'flutter run -d web-server --web-port 8080 --web-renderer html',
    url: 'http://localhost:8080',
    reuseExistingServer: true,
    timeout: 120_000,
  },
});
```

**`demos/increment-counter.spec.ts`**

```ts
import { test, expect } from '@playwright/test';

test('increment counter three times', async ({ page }) => {
  await page.goto('/');

  const increment = page.getByLabel('Increment counter');
  await increment.click();
  await increment.click();
  await increment.click();

  await expect(page.getByText('3')).toBeVisible();
  await page.waitForTimeout(1000); // hold final frame for the recording
});
```

Videos land under `ui/test-results/**/video.webm`.
