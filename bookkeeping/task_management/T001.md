**Status:** done

## Goal

Expose the existing `Predictor` class over HTTP so a web frontend can run inference without importing Python or PyTorch in the browser.

## Approach

Add a small FastAPI app that loads `Predictor` once at startup and serves inference endpoints. The browser sends HTTP requests; the Python server runs the model and returns JSON.

```
Browser  --HTTP-->  FastAPI (src/api/server.py)  -->  Predictor  -->  JSON
```

### Proposed files

| File | Purpose |
|------|---------|
| `src/api/server.py` | FastAPI app and route handlers |
| `src/model/predictor.py` | Add `predict_from_image(image, k)` for uploaded bytes/PIL images |
| `.devcontainer/environment.yml` | FastAPI deps added in T000 |

### Dependencies

## Endpoints

### `GET /health`

Liveness check.

```bash
curl http://localhost:8000/health
```

```json
{"status": "ok"}
```

### `POST /predict`

Upload a car image (multipart form). Query param `k` (default 5, max 20) sets how many predictions to return.

```bash
curl -X POST "http://localhost:8000/predict?k=5" \
  -F "file=@/path/to/car.jpg"
```

```json
{
  "predictions": [
    {"class": "kia sorento 2021", "confidence": 0.42},
    {"class": "...", "confidence": 0.18}
  ]
}
```

### `POST /predict-url`

Submit an image URL as JSON. Same behavior as `python src/test.py`.

```bash
curl -X POST http://localhost:8000/predict-url \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/car.jpg", "k": 5}'
```

Response shape matches `POST /predict`.

## Run (once implemented)

```bash
conda activate gpu-env
uvicorn src.api.server:app --host 0.0.0.0 --port 8000 --reload
```

Interactive API docs: [http://localhost:8000/docs](http://localhost:8000/docs)

```javascript
  body: formData,

const { predictions } = await res.json();

// URL
const res = await fetch("http://localhost:8000/predict-url", {
  method: "POST",
  headers: { "Content-Type": "application/json" },

```

## Error handling

Return JSON `{"detail": "..."}` with appropriate HTTP status codes. No stack traces to the client:

| Status | When |
|--------|------|
| 400 | Invalid file type, unreadable image, or unreachable URL |
| 500 | Model or unexpected inference failure |

Enable CORS for local frontend development.
