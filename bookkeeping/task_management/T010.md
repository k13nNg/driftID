### Demo script outline

Mirror the semantics-enabling pattern already used in `record-demo.spec.ts` (Flutter Web renders to canvas; click `flt-semantics-placeholder` first), then:

```ts
import { test, expect } from '@playwright/test';

test('save and reopen from history', async ({ page }) => {
  await page.goto('/');
  await page.waitForSelector('flt-semantics-placeholder', { timeout: 30_000 });
  await page.locator('flt-semantics-placeholder').dispatchEvent('click');

  // 1) Identify a car (Search tab is the default landing section, US-07).
  const upload = page.getByRole('button', { name: 'Upload image' });
  await upload.waitFor({ timeout: 15_000 });
  const [chooser] = await Promise.all([
    page.waitForEvent('filechooser'),
    upload.click(),
  ]);
  await chooser.setFiles('demos/fixtures/sample-car.jpg');
  await page.getByRole('button', { name: 'Identify' }).click();
  await expect(page.getByText('Best match')).toBeVisible({ timeout: 60_000 });

  // 2) Switch to History — entry auto-saved (US-08, US-09).
  await page.getByRole('button', { name: 'History' }).click();
  await expect(page.getByText('Best match')).toBeHidden();   // left Search tab

  // 3) Reopen the saved result (US-10) — no API call.
  await page.locator('[flt-key="history-tile"], text=ago').first().click(); // adjust to real selector
  await expect(page.getByText(/saved result/i)).toBeVisible();
  await expect(page.getByText('Best match')).toBeVisible();

  await page.waitForTimeout(2500); // hold the final frame for the recording
});
```

Selectors above are illustrative — align them with the real `Key`/`Semantics` from T005/T007/T008 (`nav-history`, `history-tile-<id>`, `saved-result`). Prefer accessible-name / role queries consistent with the existing spec.
